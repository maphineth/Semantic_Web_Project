Tolkien Gateway Knowledge Graph

Linked Data Interface for Middle-earth

About This Project

This is a Linked Data interface for the Tolkien Gateway wiki, providing semantic access to information about J.R.R. Tolkien's legendarium.

  • Content negotiation (HTML/Turtle)
  • SPARQL-powered data retrieval
  • Schema.org vocabulary alignment
  • External links to DBpedia and other KGs
  • Multilingual labels
  • Reasoning support (superclasses, owl:sameAs)
